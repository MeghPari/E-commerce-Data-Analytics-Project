# E-commerce-Data-Analytics-Project
# 📊 E-commerce Sales Analysis

This project presents an end-to-end analysis of an E-commerce Superstore dataset using **SQL** and **Python**. It focuses on extracting meaningful business insights through data querying, cleaning, exploratory data analysis (EDA), and visualization to support data-driven decision-making.

---

## 🚀 Project Overview

The objective of this project is to analyze e-commerce sales data and answer key business questions, including:

- Which products generate the highest sales and profits?
- Which regions and customer segments perform the best?
- What are the monthly sales trends?
- How do discounts affect profitability?
- Which product categories require business attention?

The project combines **SQL** for data querying and **Python** for data analysis and visualization.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/e-commerce-sales-analysis.git
```

2. Navigate to the project directory

```bash
cd e-commerce-sales-analysis
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open `e commerce sales.ipynb` and run all cells.
